Getting Started - Your First RESTful API Query

The following URL shows a sample query for Petrale sole (scientific name = Eopsetta jordani), against the trawl survey catch data, from 2010 until 2012:

json output
https://www.nwfsc.noaa.gov/data/api/v1/source/trawl.catch_fact/selection.json?filters=field_identified_taxonomy_dim$scientific_name=Eopsetta%20jordani,date_dim$year>=2010,date_dim$year<=2012 &variables=date_yyyymmdd,field_identified_taxonomy_dim$scientific_name

Click the Get button and you should receive a stream of json-formatted data:

{{ include 'ex1.html'}}

If you'd prefer a csv download of the data, change selection.json to selection.csv as follows:

csv output
https://www.nwfsc.noaa.gov/data/api/v1/source/trawl.catch_fact/selection.csv?filters=field_identified_taxonomy_dim$scientific_name=Eopsetta%20jordani,date_dim$year>=2010,date_dim$year<=2012 &variables=date_yyyymmdd,field_identified_taxonomy_dim$scientific_name

Click the Get button and you should receive a stream of csv-formatted data:
